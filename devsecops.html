Overview

This project was built to help me transition from systems and infrastructure work into DevOps and DevSecOps. The goal was not only to deploy a portfolio site, but to learn how modern delivery pipelines are built, secured, debugged, and improved step by step.

I used a self-managed VPS and domain to create a real environment where I could practice deployment, reverse proxying, SSL handling, Docker workflows, Jenkins automation, and security tooling in a practical way.

Challenge

The main challenge was moving from a manually managed static site to a more structured DevSecOps workflow. That included setting up services correctly, solving networking and proxy issues, handling Docker behavior, understanding CI/CD pipeline structure, and preparing the environment for future security scanning and monitoring.

What I implemented

1. VPS and Web Delivery

  • Configured a Linux VPS environment for hosting and testing
  • Served the portfolio site through Docker and Nginx
  • Integrated domain routing and reverse proxying

2. CI/CD Foundation

  • Installed and configured Jenkins in Docker
  • Tested Docker access from Jenkins
  • Prepared the project structure for automated build and deployment

3. Security Direction

  • Prepared for container vulnerability scanning with Trivy
  • Planned code quality integration with SonarQube
  • Used secure reverse proxy and SSL/TLS routing practices

4. Troubleshooting and Learning

  • Resolved reverse proxy and bad gateway issues
  • Debugged Docker container lifecycle problems
  • Learned the difference between static delivery and framework-based deployment
  • Improved understanding of networking, SSL modes, and container access

Tools Used

Rocky Linux Docker Nginx Jenkins Cloudflare Trivy SonarQube HTML CSS JavaScript

Outcome

This project became the foundation of my transition into DevOps. It gave me hands-on experience with containerized deployment, Jenkins setup, reverse proxy configuration, DNS and SSL troubleshooting, and the practical mindset required to build secure delivery workflows in stages.
